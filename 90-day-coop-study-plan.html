University of Waterloo Co-op Preparation

90-Day Study & Code Plan

From zero computer knowledge to co-op ready — one hour per day, every day, for twelve weeks. Designed for incoming first-year CS, SE, and CE students who need to hit the ground running.

90 Days
60 min Per day
4 Core phases
5 Portfolio projects

The One-Hour Daily Formula

Every day follows the same rhythm. The proportions shift as you progress through the 12 weeks.

0:00 Learn 12 min — Watch / read one concept. No code yet.
0:12 Code 24 min — Type along, experiment, break things.
0:36 Review 12 min — Re-type yesterday's exercise from memory.
0:48 Practice 12 min — One small problem, one flashcard deck pass.

Days marked Review / Catch-up replace the Code block with review and the Practice block with a full pass through your flashcard deck. One day off per week.

Rules for Staying on Track

Never skip two days

One rest day per week is planned. If you miss a day, do it the next day. Two misses in a row breaks the habit.

Type every example

Reading code is not learning. Writing code is. Type every example yourself. Making mistakes is the point.

Spaced repetition

Use Anki or a notebook. Re-type yesterday's exercise from memory before starting today's new material. This is built into the daily formula.

Google is your textbook

Stuck on an error? Copy the error into Google. Stack Overflow is how real developers learn. Nobody memorizes everything.

Finish before the project

If you're ahead of schedule, start the next week's project early. Projects are where learning crystallizes — don't skip them.

Talk to your code

Explain your solution out loud before coding. Rubber-ducking catches 80% of bugs before they happen. Also trains you for interviews.

Common Questions

What if I fall behind?

Shorten or skip the Review block for 2-3 days to catch up on new material. Never skip two days in a row — consistency matters more than speed.

Should I customize for CS, SE, or CE?

CS students: Follow exactly. You have the most free time in first year. SE students: Do this before arriving — you'll have zero time during 1A/1B. CE students: Do this before arriving. Your co-op starts earliest.

What comes after these 90 days?

Continue with LeetCode Medium problems (full-time during 1A/1B). Build one more project using a new tool (Docker, TypeScript, or a backend framework). Maintain your daily 30-min code habit.

I'm already in 1A and haven't started!

Start week 1 today. Cut projects from 5 to 3 (drop Bookmark App and Weather Dashboard; keep CLI Todo + the Final Project). Focus on Python, Git, and 20 LeetCode problems before 1B starts.
