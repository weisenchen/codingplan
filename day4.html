Concept

String Concatenation

You can join strings together using the + operator. This is called concatenation.

# Joining strings with +
first = "Ada"
last  = "Lovelace"
full  = first + " " + last
print(full)   # → Ada Lovelace

# You can concatenate any number of strings
msg = "Hello" + ", " + "world" + "!"
print(msg)     # → Hello, world!

⚠️ Gotcha: You cannot concatenate a string with a number directly — Python will raise a TypeError. You must convert first (more on that soon).

Concept

F-Strings (Formatted Strings)

f-strings are the modern, cleaner way to embed variables inside strings. Prefix the string with f and use {} as placeholders.

name = "Ada"
age  = 29

# Old way (hard to read)
msg1 = "My name is " + name + " and I am " + str(age) + " years old."

# F-string way (much cleaner!)
msg2 = f"My name is {name} and I am {age} years old."
print(msg2)

F-strings automatically convert values to strings — no manual str() needed inside the braces.

💡 You can even put expressions inside {}: f"{2 + 2}" → "4"

F-strings were introduced in Python 3.6. They're the recommended way to format strings in modern Python.

Concept

Type Conversion

Python has three main built-in functions for converting between types:

str()

Converts to string

str(42)    # → "42"
str(3.14)  # → "3.14"
int()

Converts to integer

int("42")  # → 42
int(3.9)  # → 3  (truncates!)
float()

Converts to decimal

float("3.14")  # → 3.14
float(42)     # → 42.0

⚠️ Common error: int("hello") raises ValueError — the string must contain a valid number.

Concept

Reading Input with input()

The input() function reads a line of text from the user. It always returns a string.

# Simple input
name = input("What is your name? ")
print(f"Hello, {name}!")

# Input + type conversion
age_str = input("How old are you? ")
age     = int(age_str)           # convert string → int
next_year = age + 1
print(f"Next year you'll be {next_year}.")

💡 Pro tip: You can wrap input() directly in a conversion function:

age = int(input("Age: "))   # read + convert in one line

The prompt string is optional but always a good idea so the user knows what to type.

Walkthrough

Building a Name-Greeting Script

Open a new file called greeting.py and follow along:

# Step 1: Ask for the user's name
name = input("What is your name? ")

# Step 2: Ask for their age (convert to int)
age = int(input("How old are you? "))

# Step 3: Ask where they're from
city = input("Which city are you from? ")

# Step 4: Build a greeting with f-strings
print(f"Hello, {name}! You are {age} years old.")
print(f"You're from {city} — that's awesome!")

# Step 5: Do a little math
years_to_30 = 30 - age
print(f"You'll be 30 in {years_to_30} years!")

Expected output:

What is your name? Alice
How old are you? 22
Which city are you from? Toronto
Hello, Alice! You are 22 years old.
You're from Toronto — that's awesome!
You'll be 30 in 8 years!
