🔑 What's a Dictionary?

A dict maps unique keys to values. Think of it like a real dictionary: you look up a word (key) and get its definition (value).

# A simple phonebook
phonebook = {
    "Alice": "555-1234",
    "Bob":   "555-5678",
    "Eve":   "555-9012"
}

# Lookup by key
print(phonebook["Alice"])  # 555-1234

Key properties

  • Keys are unique — no duplicates
  • Keys must be immutable (str, int, float, tuple)
  • Values can be anything (list, dict, int, etc.)
  • Dicts are mutable — you can change them
  • Order is preserved (Python 3.7+)

🛠️ Creating & Accessing Dicts

# Three ways to create
# 1. Curly braces
a = {"name": "Alice", "age": 25}

# 2. dict() constructor
b = dict(name="Bob", age=30)

# 3. From pairs
c = dict([
    ("x", 10), ("y", 20)
])
# Accessing values
person = {"name": "Alice", "age": 25}

# Bracket lookup (raises KeyError if missing)
print(person["name"])  # Alice

# .get() — safe, returns None
print(person.get("city"))   # None

# .get() with default
print(person.get("city", "Unknown"))
# Unknown

⚠️ [] crashes on missing keys — .get() is safer for lookups!

✏️ Adding, Updating & Deleting

# Start fresh
user = {"name": "Alice"}

# Add a new key
user["email"] = "alice@dev.com"

# Update existing
user["name"] = "A. Lice"

# or use .update()
user.update({"role": "admin", "active": True})

print(user)
# {'name': 'A. Lice', 'email': 'alice@dev.com',
#  'role': 'admin', 'active': True}
# Deleting entries
user = {"a": 1, "b": 2, "c": 3}

# del — remove a key
del user["a"]

# .pop() — remove and get value
val = user.pop("b")
print(val)  # 2

# .popitem() — remove last inserted
key, val = user.popitem()
print(key, val)  # c 3

# Clear everything
user.clear()
print(user)  # {}

🔁 Iterating Over Dictionaries

Three ways to loop — choose the right one for the job.

# .keys() — default
person = {"name": "Alice", "age": 25}

for k in person:
    print(k)
# name
# age

# .values() — just the values
for v in person.values():
    print(v)
# Alice
# 25
# .items() — both!
person = {"name": "Alice", "age": 25}

for k, v in person.items():
    print(f"{k}: {v}")
# name: Alice
# age: 25

💡 Pro tip: Use .items() when you need both — it's cleaner than indexing.

📦 Nested Dictionaries

Values can be dicts themselves — perfect for structured data.

# A richer phonebook with nested info
contacts = {
    "Alice": {
        "phone": "555-0198",
        "email": "alice@dev.com",
        "role":  "developer"
    },
    "Bob": {
        "phone": "555-0234",
        "email": "bob@dev.com",
        "role":  "designer"
    }
}

# Access nested value
print(contacts["Alice"]["email"])  # alice@dev.com

# List all developers
for name, info in contacts.items():
    if info["role"] == "developer":
        print(name)

🧰 Dictionary Patterns

Counting frequencies

text = "apple banana apple cherry banana apple"
counts = {}
for word in text.split():
    counts[word] = counts.get(word, 0) + 1

print(counts)
# {'apple': 3, 'banana': 2, 'cherry': 1}

Checking membership

user = {"name": "Alice", "active": True}

if "email" in user:
    print(user["email"])
else:
    print("No email set")

Merging dicts (3.9+)

d1 = {"a": 1, "b": 2}
d2 = {"c": 3, "d": 4}
merged = d1 | d2
# {'a': 1, 'b': 2, 'c': 3, 'd': 4}

💪 Exercise 1: Grade Tracker

Create a dict that stores student grades. Write a program that:

  1. Starts with grades = {"Alice": 85, "Bob": 92, "Eve": 78}
  2. Lets the user add a new student with their grade
  3. Prints the class average (sum of values / count)
  4. Prints the student with the highest grade
Hint: Use grades.values() to get all grades. max(grades, key=grades.get) finds the top student.

✅ Solution: Grade Tracker

# grade_tracker.py
grades = {"Alice": 85, "Bob": 92, "Eve": 78}

# Add a student
name = input("Student name: ")
score = int(input("Grade: "))
grades[name] = score

# Class average
avg = sum(grades.values()) / len(grades)
print(f"Class average: {avg:.1f}")

# Highest grade
top = max(grades, key=grades.get)
print(f"Top student: {top} ({grades[top]})")
Student name: Diana
Grade: 95
Class average: 87.5
Top student: Diana (95)

💪 Exercise 2: Word Frequency Counter

Write a program that reads a sentence and counts how many times each word appears.

  1. Ask the user for a sentence
  2. Split it into words with .split()
  3. Use a dict to count occurrences
  4. Print each word and its count, sorted by count (highest first)
Hint: .get(word, 0) + 1 is the standard counting pattern. Use sorted(counts.items(), key=lambda x: x[1], reverse=True) to sort.

✅ Solution: Word Frequency

# frequency.py
sentence = input("Enter a sentence: ")
words = sentence.lower().split()

counts = {}
for word in words:
    counts[word] = counts.get(word, 0) + 1

# Sort by frequency (highest first)
sorted_words = sorted(
    counts.items(),
    key=lambda x: x[1],
    reverse=True
)

for word, count in sorted_words:
    print(f"  {word:12s} → {count}")
Enter a sentence: the cat and the dog and the bird
the → 3
and → 2
cat → 1
dog → 1
bird → 1
