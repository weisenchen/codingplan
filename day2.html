Mental Model: The File Tree

Everything in Linux is a file. Folders are just special files that contain other files. Visualize the filesystem as an upside-down tree:

# Your filesystem looks something like this:
/
├── home/
│   └── wei/          ← your home directory (~)
│       ├── projects/
│       ├── Documents/
│       └── downloads/
├── etc/
├── usr/
└── tmp/

💡 ~ is shorthand for your home directory (here: /home/wei). / is the root — the very top.

Walkthrough: Build Your 12-Week Folder Structure

Open your terminal and type along. Each step builds on the last.

Step 1

Walkthrough: Create, Copy, Move, Delete

Now practice the core file operations on real files.

Step 4
Create a README for week-1 using touch:
cd ~/coop-prep/phase-1-python/week-1
touch README.md
ls                     # verify README.md exists
Step 5
Write something into it with echo, then view it:
echo "# Week 1: Computer Basics & Python" > README.md
cat README.md
Step 6
Copy the README to week-2, then rename:
cp README.md ../week-2/
mv ../week-2/README.md ../week-2/README-WEEK2.md
ls ../week-2/
Step 7
Create a scratch file, then delete it:
touch scratch.txt
rm scratch.txt
ls                     # scratch.txt should be gone
8 / 13

Walkthrough: Viewing File Contents

Not all files are small. Learn to peek inside without opening an editor.

Step 8
Use cat for short files — prints everything at once:
cat ~/coop-prep/phase-1-python/week-1/README.md
Step 9
Use less for long files — scroll line by line. Press q to quit:
less /etc/passwd          # scroll with ↑↓ or PageUp/PageDown, q to quit
Step 10
See your full folder tree with tree:
tree ~/coop-prep

💡 Pro tip: cat is for short files; less is for long files or logs. Get in the habit of less.

Exercise 1: Organize Your Downloads

📝 Task

Create a folder structure to organize downloaded materials for this course:

coop-prep/
├── downloads/
│   ├── videos/
│   ├── code-samples/
│   └── references/

Use mkdir -p to create all three subfolders in one command. Then verify with tree.

Need a hint? 
mkdir -p ~/coop-prep/downloads/{videos,code-samples,references}
tree ~/coop-prep/downloads

💡 -p creates parent directories if needed. {a,b,c} is brace expansion — creates all three at once.
