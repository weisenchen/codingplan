✍️ Exercise 1: Array Transform Pipeline

Re-type this from memory. Close your laptop, look away, then write it.

const users = [
  { name: "Alice", age: 24, active: true },
  { name: "Bob", age: 17, active: false },
  { name: "Carol", age: 31, active: true },
];

// Get names of active users over 18, uppercased
const result = users
  .filter(u => u.active && u.age >= 18)
  .map(u => u.name.toUpperCase());

// Expected: ["ALICE", "CAROL"]
console.log(result);
💡 How to practice

1. Read the code once → 2. Hide the panel (scroll up) → 3. Type it from memory → 4. Compare. Repeat until flawless.

✍️ Exercise 2: Build a List from Data

This pattern appears in every front-end project. Type it until it's automatic.

const items = ["Learn JS", "Build projects", "Apply to co-op"];
const list = document.createElement("ul");

items.forEach(text => {
  const li = document.createElement("li");
  li.textContent = text;
  list.appendChild(li);
});

// Attach to the page
document.body.appendChild(list);
// Note: el.append() can take multiple nodes/strings too!

Variation: use document.createDocumentFragment() for performance when adding many items.

✍️ Exercise 3: reduce() Patterns

The Swiss Army knife of array methods. Practice the two most common patterns.

// Pattern 1: Sum
const numbers = [3, 7, 2, 9, 5];
const sum = numbers.reduce((acc, n) => acc + n, 0);
// sum → 26

// Pattern 2: Group by property
const people = [
  { name: "Alice", role: "dev" },
  { name: "Bob", role: "design" },
  { name: "Carol", role: "dev" },
];
const grouped = people.reduce((acc, p) => {
  (acc[p.role] ??= []).push(p);
  return acc;
}, {});

Try typing both patterns three times without looking.

🧪 Practice Exercises

Try these on your own. Click "Show solution" after attempting each.

Exercise A: getAdults(people)

Write a function that takes an array of {name, age} objects and returns an array of names of people 18+.

Show solution 
const getAdults = people =>
  people.filter(p => p.age >= 18).map(p => p.name);
Exercise B: Toggle Class

Select an element by #myBtn. When clicked, toggle the class "active" on it.

Show solution 
document.getElementById("myBtn")
  .addEventListener("click", e =>
    e.currentTarget.classList.toggle("active")
  );
Exercise C: Shopping Cart Total

Given cart = [{item:"shirt", price:25,qty:2}, {item:"hat",price:15,qty:1}], compute the total using reduce().

Show solution 
const total = cart.reduce((sum, item) =>
  sum + item.price * item.qty, 0);
// total → 65
