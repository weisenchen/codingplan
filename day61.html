① Two Sum

Given an array of integers nums and a target target, return indices of the two numbers that add up to target.

def twoSum(nums, target):
    seen = {}                 # val → index
    for i, n in enumerate(nums):
        diff = target - n
        if diff in seen:
            return [seen[diff], i]
        seen[n] = i
    return []

Key insight: For each number, check if its complement already exists in the hash map. One pass — O(n) time, O(n) space.

Edge cases: No solution (return []), negative numbers, duplicates.

💡 Tip: Store the value as key, index as value. Don't forget to check diff in seen before inserting.

② Reverse Linked List

Reverse a singly linked list. Return the new head.

def reverseList(head):
    prev = None
    curr = head
    while curr:
        nxt = curr.next      # save next
        curr.next = prev     # reverse pointer
        prev = curr          # move prev forward
        curr = nxt           # move curr forward
    return prev

Three-pointer technique: prev, curr, nxt. Walk the list, flip each arrow backward.

Edge cases: Empty list (head is None), single node.

Recursive version also works — but iterative is faster and less stack-prone.

💡 Tip: Trace with a 3-node list on paper. Watch how prev ends up at the old tail (the new head).

③ Valid Parentheses

Given a string containing (){}[], determine if it's valid — every open bracket has a matching close in the correct order.

pairs = {')': '(', '}': '{', ']': '['}

def isValid(s):
    stack = []
    for ch in s:
        if ch in pairs:          # closing bracket
            if not stack or \
               stack.pop() != pairs[ch]:
                return False
        else:
            stack.append(ch)    # opening bracket
    return not stack

Stack invariant: Push opens, pop and match on closes. Empty stack at the end = valid.

Edge cases: Single char "(", mismatched "(]", nested "{[]}".

💡 Tip: Use a dict to map close → open for clean matching. not stack at the end handles unclosed brackets.

④⑤ Max Depth + FizzBuzz

Max Depth of Binary Tree

def maxDepth(root):
    if not root:
        return 0
    left = maxDepth(root.left)
    right = maxDepth(root.right)
    return 1 + max(left, right)

Post-order traversal. Empty node = depth 0. Add 1 for the current node. O(n).

BFS alternative: Level-order with a queue — also O(n).

FizzBuzz

def fizzBuzz(n):
    res = []
    for i in range(1, n + 1):
        s = ""
        if i % 3 == 0: s += "Fizz"
        if i % 5 == 0: s += "Buzz"
        if not s: s = str(i)
        res.append(s)
    return res

String concatenation avoids duplicate divisibility checks. Clean and extensible.

Edge case: n = 0 → empty list.
