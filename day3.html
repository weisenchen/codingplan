Install & Verify Python

Open your terminal and type these commands one at a time.

Step 1 — Check if Python is already installed
Run this in the terminal:
python3 --version
Expected output: Python 3.10.x or similar. If you see this, you're good.
Step 2 — If you get "command not found"
Install Python from python.org/downloads. Download the latest Python 3 installer for your OS and run it.
Step 3 — Verify again
After installing, run this to confirm:
python3 --version
# Should print something like: Python 3.13.x

💡 On Linux (Ubuntu), you can also install via: sudo apt install python3

Variables: Python's Label Maker

A variable is a named label for a value. Think of it as a sticky note you put on a box.

Variable
message
The name — your label
Value
"Hello, world!"
The data stored inside
Assignment
=
Puts the value into the variable
Read it back
print(message)
Use the name to get the value
message = "Hello, world!"     # assign
print(message)              # read → Hello, world!

💡 Variable names are case-sensitive. message ≠ Message

Three Basic Data Types

Every value in Python has a type. The type determines what you can do with it.

Type Example Used for
int42, -3, 0Whole numbers, counters, indices
float3.14, -0.5, 2.0Decimals, measurements, averages
str"hello", 'a'Text, names, messages, output

Try this in hello.py — replace everything, then run it:

name = "Waterloo"
year = 2026
pi = 3.14159
print(name)
print(year)
print(pi)

Let's build a program that introduces itself — and you.

📝 Exercise 1: About Me

Create a script about_me.py that stores your name, age (as int), height in meters (as float), and favorite food (as str) in variables. Print them all in one sentence.

Hi, I'm Weisen. I'm 18 years old, 1.75m tall, and I love pizza.
