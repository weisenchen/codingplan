Concept

for item in list: — Iterating over a List

A for loop lets you visit every element in a list, one at a time, without managing an index.

fruits = "apple", "banana", "cherry"]

for fruit in fruits:
    print(fruit)

How it reads: "For each fruit in the list fruits, do something."

The variable name (fruit) is your choice — use singular nouns for readability.

 
Examples

for Loop Patterns

Filter in a loop

numbers = [3, 8, 1, 12, 5]

for n in numbers:
    if n > 5:
        print(n, "is big")

Build a new list

words = ["cat", "dog", "bird"]
caps = []

for w in words:
    caps.append(w.upper())

print(caps)

Key insight: The loop body runs once per item. After the last item, execution continues after the loop.

 
Walkthrough

Sum a List of Numbers

Let's add up every number in a list — the classic loop example.

scores = [85, 92, 78, 95, 88]
total = 0        # 1. Start at zero

for s in scores:  # 2. Loop through each score
    total += s    # 3. Add current score to total

print(total)    # 4. Output: 438

Step by step:

  1. Initialize total = 0 (the accumulator)
  2. Each iteration: total = total + s
  3. After the last item, total holds the sum

This pattern works for averages, products, concatenation — anything that accumulates.

 
Concept

range() — Looping with an Index

Use range(n) when you need the position, not just the value.

colors = ["red", "green", "blue"]

for i in range(len(colors)):
    print(i, colors[i])

→ 0 red
1 green
2 blue

# range(start, stop, step)
for i in range(2, 10, 3):
    print(i)

→ 2 5 8

Remember: range(n) gives 0, 1, 2, ..., n-1. Combine with len(list) for index-based access.

 
Concept

while Loops — Repeat Until False

A while loop keeps running as long as its condition is True.

count = 1

while count <= 5:
    print("Count:", count)
    count += 1

→ prints 1 through 5

When to use while

  • You don't know how many iterations you need
  • Waiting for user input (while True/break)
  • Reading data until the end of a stream
  • Game loops, simulation timesteps

⚠️ Danger: If the condition never becomes False, you get an infinite loop. Always make sure your condition will eventually change!

 
Concept

break & continue — Loop Control

break — exit the loop

while True:
    cmd = input("> ")
    if cmd == "quit":
        break    # ← stops the loop
    print("You said:", cmd)

The classic "repeat until quit" menu pattern.

continue — skip this iteration

for n in range(1, 10):
    if n % 2 == 0:
        continue  # ← skip evens
    print(n)

Skip even numbers — prints: 1 3 5 7 9

Rule of thumb: Use break to exit early. Use continue to skip one iteration. Both work in for and while loops.

 
Comparison

for vs while — When to Use Each

Use for when…

  • You know the collection you're iterating over
  • You want to process each element in a list
  • You want clean, concise code
  • You need a fixed number of iterations (range)
# "For each item" — very clear
for name in names:
    print(name)

Use while when…

  • The number of iterations is unknown
  • You're waiting for a condition to become false
  • You need a persistent loop (e.g., game loop)
  • You're not iterating over a sequence at all
# "Keep asking until" — more flexible
guess = ""
while guess != "secret":
    guess = input("Guess: ")

Most of the time in Python, reach for for first. It's safer (no infinite loops) and more readable.

Exercises — Try It Yourself

Exercise 1 · Beginner
Find the maximum: Given nums = [7, 2, 9, 4, 1], write a loop to find and print the largest number. Do not use max().
Hint: Start a variable at 0 and update it when you see a bigger number. What if all numbers are negative?
Exercise 2 · Intermediate
Guess the number: Use a while loop to create a guessing game. Pick a secret number (7). Keep asking the user to guess until they get it right. Print "Too high" or "Too low" after each wrong guess.
Hint: Use while True and break when they guess correctly. Cast input() to int.
Exercise 3 · Challenge
Count vowels: Given text = "hello world", write a loop that counts how many vowels (a, e, i, o, u) appear in the string. Print the total.
Hint: A string is iterable — for ch in text: works! Use in with a string of vowels: if ch in "aeiou".

Solutions

Exercise Solutions

Ex 1: Find max

nums = [7, 2, 9, 4, 1]
max_num = nums[0]   # start with first

for n in nums:
    if n > max_num:
        max_num = n
print(max_num)  # → 9

Ex 2: Guess the number

secret = 7
while True:
    guess = int(input("Guess: "))
    if guess == secret:
        print("Correct!")
        break
    elif guess < secret:
        print("Too low")
    else:
        print("Too high")

Ex 3: Count vowels

text = "hello world"
vowels = "aeiou"
count = 0

for ch in text:
    if ch in vowels:
        count += 1
print(count)  # → 3 (e, o, o)
 
Recap

Key Takeaways

🔁
for item in list: visits each element
📊
Accumulator pattern: total += value
🔄
while loops run until a condition is False
🚦
break exits; continue skips ahead

Three things to remember:

  1. Prefer for over while when iterating — it's safer and clearer
  2. Always ensure while conditions will eventually become False
  3. Accumulator pattern: initialize before the loop, update inside
