🧱 Project Overview

A CLI todo list — add tasks, mark them done, delete them, save to a JSON file

📁 Project Structure

One file. One JSON store. Four functions.

File layout

todo.py          # your script
tasks.json       # created on first run

JSON format

[
  {
    "id": 1,
    "task": "Buy groceries",
    "done": false
  },
  ...
]

Each task has three fields: id (int), task (str), done (bool).

🛠 Step 1 — Load & Save Tasks

These two helpers wrap JSON file I/O so the rest of the program is clean.

import json
import os

DATA_FILE = "tasks.json"

def load_tasks():
    """Load tasks from JSON file. Returns [] if file missing."""
    if not os.path.exists(DATA_FILE):
        return []
    with open(DATA_FILE, "r") as f:
        return json.load(f)

def save_tasks(tasks):
    """Save tasks list to JSON file."""
    with open(DATA_FILE, "w") as f:
        json.dump(tasks, f, indent=2)
🔑 Key idea: json.load() reads the file into a Python list. json.dump() writes it back. indent=2 makes the file human-readable.

➕ Step 2 — Add a Task

Takes a task description, assigns the next ID, appends it, saves.

def add_task(description):
    """Add a new task. Returns the new task dict."""
    tasks = load_tasks()
    # Next ID: max existing + 1, or 1 if empty
    new_id = max((t["id"] for t in tasks), default=0) + 1
    task = {"id": new_id, "task": description, "done": False}
    tasks.append(task)
    save_tasks(tasks)
    return task
🔑 Key idea: max(..., default=0)+1 gives us auto-incrementing IDs — no collisions.

📋 Step 3 — List, Done & Delete

📄 List

def list_tasks(tasks):
    if not tasks:
        print("No tasks yet!")
        return
    for t in tasks:
        status = "✓" if t["done"] else " "
        print(f"[{status}] {t['id']}. {t['task']}")

✅ Mark Done

def done_task(task_id):
    tasks = load_tasks()
    for t in tasks:
        if t["id"] == task_id:
            t["done"] = True
            save_tasks(tasks)
            return t
    print(f"Task {task_id} not found.")

🗑 Delete

def delete_task(task_id):
    tasks = load_tasks()
    new_tasks = [t for t in tasks
                 if t["id"] != task_id]
    if len(new_tasks) == len(tasks):
        print(f"Task {task_id} not found.")
        return
    save_tasks(new_tasks)
    print(f"Deleted task {task_id}.")

🏗 Step 4 — The Main Loop

A simple REPL that reads commands and dispatches to our functions.

def main():
    while True:
        cmd = input("todo> ").strip().lower()
        if cmd == "exit" or cmd == "quit":
            break
        elif cmd == "add":
            desc = input("Task description: ")
            add_task(desc)
            print("Task added!")
        elif cmd == "list":
            list_tasks(load_tasks())
        elif cmd.startswith("done"):
            # e.g. "done 3"
            parts = cmd.split()
            if len(parts) == 2 and parts[1].isdigit():
                done_task(int(parts[1]))
        elif cmd.startswith("delete"):
            parts = cmd.split()
            if len(parts) == 2 and parts[1].isdigit():
                delete_task(int(parts[1]))
        else:
            print("Commands: add, list, done <id>, delete <id>, exit")

if __name__ == "__main__":
    main()
🔑 Key idea: The if __name__ == "__main__" guard lets you import this file without running it.
